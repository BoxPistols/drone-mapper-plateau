## 1. 場所検索

ヘッダー中央の検索バーに地名・施設名・住所を入力すると、カメラが自動的にその場所に移動します。

- **検索エンジン**: OpenStreetMap Nominatim（無料・APIキー不要）
- **対象エリア**: 日本国内（`countrycodes=jp` で絞り込み）
- **デバウンス**: 400ms（タイピング中のAPI呼び出しを抑制）

### 使い方
1. ヘッダーの検索バーをクリック
2. キーワードを入力（例：「浅草寺」「東京都庁」「仙台城」）
3. ドロップダウンから候補を選択 → カメラが飛行

## 2. 3D都市モデル表示

左サイドバー「マップ」タブ > 「3D都市モデル」セクションから都市を切り替えます。

| 都市 | 都道府県 | LOD | データ年度 |
|------|---------|-----|-----------|
| 台東区 | 東京都 | LOD2 | 2024 |
| 港区 | 東京都 | LOD2 | 2023 |
| 仙台市（青葉区） | 宮城県 | LOD2 | 2024 |
| 加賀市 | 石川県 | LOD2 | 2024 |
| 沼津市 | 静岡県 | LOD3（テクスチャあり） | 2023 |
| 広島市（中区） | 広島県 | LOD2 | 2024 |
| 福岡市（中央区） | 福岡県 | LOD2（テクスチャあり） | 2024 |

### LODについて
- **LOD1**: 押し出し形状（高さあり、屋根形状なし）
- **LOD2**: 屋根形状あり（最も一般的）
- **LOD3**: 開口部（窓・扉）まで表現（高精細）

### 建物クリック
3D建物をクリックすると右上に属性パネルが表示されます。

| 表示項目 | 内容 |
|---------|------|
| 建物ID | GML識別子（`gml_id`）|
| 高さ | 計測高さ（`measuredHeight`、単位m）|
| 地上階数 | 階数（`storeysAboveGround`）|
| 用途 | 建物用途コード |
| 建築年 | 建設年（`yearOfConstruction`）|

> **注意**: 属性の有無はデータセット・LODによって異なります。LOD1では高さ以外が含まれない場合があります。

## 3. ピン設置

任意の地点にポイントマーカーを設置します。

### 操作手順
1. マップツールバーの「ピン」ボタンをクリック（青くハイライト）
2. マップ上の任意の地点をクリック → ピンが設置される
3. サイドバー「マップ」タブのピン一覧からクリックして展開
4. 名前・メモを編集、または「×」で削除

### 保存
localStorage に自動保存されます。ブラウザを閉じても残ります。

---

## 4. ゾーン描画（飛行エリア管理）

飛行計画エリア・飛行禁止エリアなどをポリゴンで描画します。

### 操作手順
1. マップツールバーの「ゾーン」ボタンをクリック
2. マップ上をクリックして頂点を追加（3点以上）
3. **ダブルクリック** または ツールバーの「確定」ボタンでポリゴン確定
4. ゾーン名を入力
5. ゾーン種別を選択

### ゾーン種別

| 種別 | 用途 | 色 |
|------|------|---|
| 計画エリア | 飛行予定区域 | 青 |
| 飛行禁止 | 立入禁止・規制区域 | 赤 |
| 注意エリア | 要注意区域（電線・建物密集など）| 黄 |
| 完了済み | 飛行済み区域 | 緑 |

## 5. 飛行計画

ウェイポイントベースの飛行ルートを作成・管理します。

### 計画の作成
1. サイドバー「計画」タブ → 「+ 新規作成」
2. 計画名・パイロット・機体・計画日・最大高度を入力
3. 「マップ編集」ボタンをクリック（ウェイポイント追加モードに切替）
4. マップ上をクリックしてウェイポイントを順番に配置
5. 「完了」ボタンでウェイポイント追加モードを終了

### ウェイポイント設定

各ウェイポイントをクリックして展開すると詳細設定できます：

| パラメータ | 内容 | デフォルト |
|-----------|------|-----------|
| 高度（AGL） | 地上高（Above Ground Level） | 50m |
| 速度 | 飛行速度 | 5 m/s |
| アクション | 該当WP到達時の動作 | 無し |
| 停止時間 | ホバリング時間（アクション＝ホバーの場合） | 5秒 |

### ウェイポイントアクション

| アクション | 内容 |
|-----------|------|
| 無し | 通過するのみ |
| 撮影 | 静止画撮影トリガー |
| 録画開始 | 動画録画開始 |
| 録画停止 | 動画録画停止 |
| ホバリング | 指定秒数停止 |

> **注意**: アクションは計画上の記録であり、実機への自動送信機能はありません。

### ステータス管理

| ステータス | 説明 |
|-----------|------|
| 草稿 | 作成中・未承認 |
| 承認済 | 上長・安全管理者の承認済み |
| 完了 | 飛行完了 |

## 6. 飛行シミュレーション

作成した飛行計画を3Dマップ上でアニメーション再生します。

### 起動
計画詳細の「▶ シミュレート」ボタン（ウェイポイントが2点以上必要）

### 操作

| 操作 | 説明 |
|------|------|
| ▶ / ⏸ | 再生 / 一時停止 |
| 速度ボタン | 1× / 2× / 5× / 10× |
| シークバー | 任意の地点にジャンプ |
| リセット | 最初のウェイポイントに戻る |
| × | シミュレーション終了 |

### 表示要素

- **青い点**: ドローンの現在位置
- **青いライン**: 飛行済みのパス
- **黄色い楕円**: カメラのFOV（視野角）の地面投影
- **AGL表示**: 現在の地上高
- **WP番号**: 現在通過中のウェイポイント番号

### 所要時間の計算
ウェイポイント間の3D距離 ÷ 各区間の速度（m/s）の合計。ホバリング時間は別途加算されません（現バージョン）。

## 7. 飛行記録

実際の飛行結果を記録・管理します。

### 作成方法
- 「記録」タブ → 「+ 新規記録」
- または飛行計画詳細から「+ 飛行記録作成」（計画情報を引き継ぎ）

### 記録項目

| 項目 | 必須 | 内容 |
|------|------|------|
| 記録名 | ○ | 飛行記録の名称 |
| パイロット | ○ | 操縦者氏名 |
| 飛行日 | ○ | 実施日 |
| 開始時刻 | - | 離陸時刻 |
| 終了時刻 | - | 着陸時刻 |
| 天候 | - | 晴れ/曇り/薄曇り/小雨/強風注意 |
| 最大風速 | - | 実測値（m/s）|
| 実測最大高度 | - | 実際の最高高度（m）|
| 飛行距離 | - | 総飛行距離（m）|
| 関連計画 | - | 対応する飛行計画との紐づけ |
| 特記事項 | - | 自由記述 |
| ステータス | ○ | 予定/飛行中/完了/キャンセル |

---

## 8. 撮影データ管理

飛行で取得した写真・動画ファイルを管理します。

### アップロード方法
- 「データ」タブ → 「+ 追加」ボタン
- またはドロップエリアにファイルをドラッグ&ドロップ

### 対応形式
- 写真: JPEG, PNG, WebP など（ブラウザ対応形式）
- 動画: MP4, MOV など（ブラウザ対応形式）

### 表示
- 写真: サムネイルプレビュー
- 動画: プレースホルダーアイコン
- 飛行記録・計画別にグループ表示

> **現バージョンの制限**: ファイルはブラウザメモリ上で処理されます。大きなファイルはページリロードで消えます。永続化にはサーバーサイドストレージが必要です。

---

## データの永続化

| データ種別 | 保存先 | 備考 |
|-----------|--------|------|
| ピン | localStorage | 永続 |
| ゾーン | localStorage | 永続 |
| 飛行計画 | localStorage | 永続 |
| 飛行記録 | localStorage | 永続 |
| 撮影データ（メタデータ） | localStorage | 永続 |
| 撮影データ（ファイル本体） | メモリ | ページリロードで消える |
| シミュレーション状態 | メモリ | 一時的 |
| 都市選択 | localStorage | 永続 |
